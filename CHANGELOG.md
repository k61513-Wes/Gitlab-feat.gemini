## 如何填寫本文件

每次有任何改動，不論大小，都在對應版本下方的表格新增一行：

```
| [序號] | [YYYY-MM-DD] | [版本] | [類型] | [異動檔案] | [說明] |
```

**類型說明：**

| 類型 | 說明 |
|------|------|
| feat | 新增功能 |
| fix | 修復 Bug |
| docs | 文件異動（無程式碼改動） |
| refactor | 程式碼重構（無功能改動） |
| style | 格式調整（縮排、版號顯示等） |
| chore | 設定、工具、Git 操作等雜項 |
| test | 測試相關 |

若同一次改動涉及多個檔案，每個檔案獨立一行記錄。
